Introduction

The goal of this project is to make 3 predictive regression models which predict wind turbine power from wind speed.

Predicting power values from wind speed is very important for electricity production. For instance, average wind speed in an area can be calculated and it can be determined if it is worthwhile to construct a wind turbine in that area by calculating the predicted power output.

The three regression models that will be used in this project are the linear regression model, the sigmoid regression model and the polynomial regression model.

Linear regression

This project will use the the linear_model function from the sklearn package to perform the linear regression

This function takes the input values (x) and generates a predicted value (y) using the formula y = ax + b, where a and b are coefficients that are fit using Ordinary Least Squares.

Ordinary Least Squares works by choosing coefficients that minimise the difference between the observed and predicted values of the dataset 1. The observed values are the actual target values of y and the predicted values are the values generated by the formula y = ax + b.

Preprocessing

First the data is imported and preprocessed. The preprocessing consists in splitting up the data into feature data (x values) and labels (y values) and rehsaping the X values as the LinearRegression() from linear_model function does not take a 1D array for the X values. First, a scatter plot of power versus speed is shown:

Sigmoid regression

As stated in the Preprocessing subsection of the Linear Regression section, the function resembles a sigmoid function. A sigmoid function resembles an S shaped curve. It often appears as the integral of various probability density functions including the normal distribution 5.

A common sigmoid function is the logistic function. The logistic function has been used in epidemiological studies of viral outbreaks, for example the Covid-19 epidemic 6.

To create a regression model using the logistic function, the original dataset must be transformed and then linear regression is used on the transformed dataset.

Preprocessing

The logistic function is of the form A/(1+exp(ax + b)) where A, a and b are constants. A in this instance is the limiting value as x->. In this case, judging by the graph in the Preprocessing subsection of the Linear Regression section, it is approximately 100. a and b will be determined by the linear regression fit after the data is transformed.

The first transformation gets rid of rows where the power is 0 Watts or greater than or equal to 100 Watts. This is because the second transformation involves a fraction in a log expression. The fraction cannot have 0 in the denominator and the log expression cannot be 0 or less.

Polynomial regression

The sigmoid regression seems to be a good fit for the data. However, it could be the case that the relationship is polynomial, i.e., the relationship between the output values y and the input values x is of the form y = anxn + anxn-1... + a1x + a0 7 where the vector a is a vector of constants and the vector x is a vector of input values of dimension n.

An example of a real life application is the design of bridges 8. Engineers use polynomial functions to map the curvature of these bridges as polynomial functions are often curved functions.

Creating a polynomial regression on the data can be achieved in the same way as the sigmoid regression i.e. transforming the data and applying a linear regression on the resulting transformed data.

Preprocessing

The preprocessing from the linear regression model is repeated in this step to remove the outliers and reshape the data for the regression function.

Algorithm

The PolynomialFeatures function from sklearn.preprocessing is used to create a transformed version of the dataset for the polynomial regression. This works by creating a new column of data which is the original dataset raised to the power of a set exponent, e.g., if the original dataset is 1, 2, 3 and the exponent is set to 2, the new dataset is 1, 4, 9 9.

If the exponent is 3, then two new columns are added: 1, 4, 9 and 1, 8, 27 which correspond to x2 and x3 respectively.

The make_pipeline function from sklearn.pipeline is used to create a pipeline between the transformation and the linear regression. The degree of the polynomial function is chosen to be 15. After this, the data is split using train_test_split and the polynomial regression is applied as before:

In [128]:
from sklearn.preprocessing import PolynomialFeatures
from sklearn.pipeline import make_pipeline
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

polyreg=make_pipeline(PolynomialFeatures((15)),LinearRegression())

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=0)

polyreg.fit(X_train, y_train)

Discussion ¶

The sigmoid regression model produces a coefficient of variation that is 14% less than the linear regression model so the sigmoid regression model is more accurate. In addition, the linear regression model is above the threshold value of 25% whereas the sigmoid regression model is 13% less than the threshold value so the sigmoid regression model meets the criterion for an accurate regression model.

However, the polynomial regression had a coefficient of variation that was more than 3% less than the value for the sigmoid regression model, which by that metric makes it the most accurate model out of the 3. The issue with the model is that it is extremely complex and overly flexible. The number of terms in the equation was 15 which is unusually large, it is much more common to have 2 or 3 terms in the polynomial regerssion. Polynomial expressions of this degree often leads to very strange shapes and overfitting but in this case the cross validation should have mitigated the negative effects of overfitting.

In this project, the 3 models used to create the regression models were based on a simple algebraic and logarithmic functions. Further work will be done in the "Machine Learning and Statistics" project where more complex regression models are developed using decision trees and neural networks.
